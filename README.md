# Nến Thơm Demo

Demo web bán nến thơm.

## Cài đặt

```bash
npm install
```

## Chạy dev

```bash
npm run dev
```

## Build production

```bash
npm run build
npm run preview
```

## Thư mục chính
- `client/` : Frontend React/Vite
- `public/` : Ảnh, favicon, ...
- `data/` : Dữ liệu mẫu
